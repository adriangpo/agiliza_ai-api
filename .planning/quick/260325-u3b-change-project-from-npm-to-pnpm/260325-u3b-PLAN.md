**Makefile** — replace `npx` calls with `pnpm dlx` (or just run the bins directly since they are in node_modules/.bin via pnpm):

Change:
```makefile

	npx eslint . --max-warnings 0

	npx eslint . --fix --max-warnings 0

	npx tsc --noEmit
```

To (use `pnpm exec` to run local bins without npx):
```makefile
lint:
	pnpm exec eslint . --max-warnings 0

lint-fix:
	pnpm exec eslint . --fix --max-warnings 0

typecheck:
	pnpm exec tsc --noEmit
```

**lefthook.yml** — replace all `npx` with `pnpm dlx` (for staged-file invocations, `pnpm dlx` resolves from the project's installed packages):

```yaml
pre-commit:
  parallel: false
  commands:
    lint:
      glob: '*.{ts,js}'
      run: pnpm exec eslint --max-warnings 0 {staged_files}
    format:
      glob: '*.{ts,js}'
      run: pnpm exec prettier --write {staged_files} && git add {staged_files}
    typecheck:
      run: pnpm exec tsc --noEmit

commit-msg:
  commands:
    commitlint:
      run: pnpm exec commitlint --edit {1}
```

**CI workflow** (`.github/workflows/ci.yml`) — four jobs all need updating:
