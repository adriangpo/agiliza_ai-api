## Changes Made

### Task 1: engines field
`package.json` now declares `"engines": { "node": ">=24" }` immediately after `"license"`. This makes the Node.js 24 constraint from CLAUDE.md machine-enforceable — npm/yarn/pnpm will warn or error if the installed Node.js version is too old.

### Task 2: limiter API docs
Both `app/shared/middleware/rate_limit_middleware.ts` and `start/limiter.ts` previously contained positive examples of `middleware.throttle('submissions')` and `router.post('/reports').use(middleware.throttle(...))`. This API does not exist in `@adonisjs/limiter` v3. The files now:
- Explicitly state that `middleware.throttle()` does NOT exist in v3
- Document the correct inline `limiter.use({ requests, duration }).attempt(key, fn)` pattern
- Align with the note already present in `start/kernel.ts`
